# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/),
und dieses Projekt hält sich an [Semantic Versioning](https://semver.org/lang/de/).

## [1.1.0] - 2025-11-13

### Hinzugefügt
- 20 neue Block Patterns für moderne SaaS-Webseiten
- **Hero Patterns:** Video Background Hero, Animated Gradient Hero, Hero mit Statistiken
- **About Patterns:** Team Grid, Company Values, Company Timeline
- **Feature Patterns:** Integration Showcase, Feature Comparison, Process Steps
- **Testimonial Patterns:** Review Cards, Customer Logos
- **Statistics Patterns:** Metrics Dashboard
- **Gallery Patterns:** Screenshot Showcase
- **CTA Patterns:** Newsletter Signup, Demo Booking
- **FAQ Patterns:** Accordion FAQ mit ausklappbaren Antworten
- **Text Patterns:** Highlight Boxes, Quote Block
- **Posts Patterns:** Blog Grid mit 3 Spalten
- **Banner Patterns:** Limited Time Offer Banner
- Insgesamt jetzt 30+ professionelle Block Patterns verfügbar

## [1.0.0] - 2025-11-13

### Hinzugefügt
- Initiales Release des SaaS Enterprise Themes
- Vollständige WordPress Block Theme Struktur
- Theme.json mit umfassender Farbpalette und Typography
- 4 Basis-Templates (index, page, single, 404)
- Header und Footer Template Parts
- Grundlegende Block Patterns (Hero, Pricing Table)
- JavaScript-Features (Back-to-Top, Reading Progress, Mobile Menu, Cookie Consent)
- Custom CSS für Animationen und Hover-Effekte
- Responsive Design mit Mobile-First Ansatz
- Google Fonts Integration (Poppins und Inter)
- 12 Pattern-Kategorien für verschiedene Sektionen
- GPL v2 Lizenz
- Deutsche Lokalisierung

### Geplant für zukünftige Versionen
- Weitere 60+ Block Patterns
- Zusätzliche Template Parts (verschiedene Header- und Footer-Varianten)
- Erweiterte JavaScript-Features
- Performance-Optimierungen
- Accessibility-Verbesserungen
- Weitere Dokumentation und Tutorials
